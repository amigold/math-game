# 🎮 Math Orb Collector

A fun, interactive math game designed for first-grade students to practice addition through collecting colorful rewards!

## 🎯 Features

- **Three Types of Orbs:**
  - 🔵 Regular Orbs: Earn emoji collectibles
  - ⭐ Magic Orbs (every 5 problems): Earn special face emojis
  - 🌈 Rainbow Orbs (every 10 problems): Complete timed challenges for picture rewards!

- **Progressive Difficulty:** Problems get harder as you progress
- **Challenge Mode:** 2-minute timed challenges with picture rewards
- **Collections System:** Three separate collections that persist across sessions
- **Bonus System:** Collect the same picture 4 times to earn a magic face emoji!

## 🎨 How to Play

1. Solve addition problems by typing your answer
2. Press Enter or click "Check" to submit
3. Correct answers unlock rewards!
4. Watch for special orbs:
   - Every 5 problems: Golden magic orb with sparkles
   - Every 10 problems: Rainbow challenge orb

## 🔧 Customization

To add your own pictures as rewards:
1. Add image files to the `/pictures` folder
2. Update `pictures.js` with the filenames
3. Pictures will appear as rewards in challenge mode!

## 💾 Technology

- Pure HTML5, CSS3, and JavaScript
- No frameworks or dependencies
- Web Audio API for sound effects
- localStorage for data persistence
- Fully responsive design
